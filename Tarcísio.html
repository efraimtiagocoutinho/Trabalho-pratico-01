GitHub


GitHub é uma rede social para programadores. Github permite participar em colaboração por bifurcação projetos, o envio de e puxando pedidos e acompanhamento do desenvolvimento.
GitHub é um serviço de hospedagem baseado na web para projetos de desenvolvimento de software que usam o Git controle de revisão do sistema.
 GitHub oferece dois planos pagos para repositórios privados e contas gratuitas para open source projetos. GitHub foi lançado em abril de 2008. Desenvolvimento da plataforma GitHub começou em 19 de outubro de 2007.

Este site possui funcionalidades de uma rede social como feeds, followers, wiki e um gráfico que mostra como os desenvolvedores trabalham as versões de seus repositórios.

O GitHub também possui um serviço tipo pastebin chamado Gist.
Em uma palestra no Yahoo! sede em 24 de fevereiro de 2009, os membros da equipe GitHub anunciou que durante o primeiro ano que o GitHub foi on-line, que acumulou 46.000 repositórios públicos, 17.000 deles no mês anterior sozinho.
 Naquela época, cerca de 6.200 repositórios foi bifurcada pelo menos uma vez e 4.600 mescladas. Em 5 de Julho de 2009, um Github Blog pós anunciaram que chegaram aos usuários 100.000 marcar.
Em outra conversa entregue no Yahoo! em 27 de julho de 2009, Tom Preston-Werner anunciou que o número subiu para 90.000 repositórios públicos únicos, 12.000 tendo sido bifurcadas pelo menos uma vez, para um total de 135.000 repositórios.
  Em julho de 2010, GitHub anunciou que abriga 1 milhão de repositórios.  Em abril de 2011, GitHub anunciou que está hospedando 2 milhões de repositórios.

Em 21 de Setembro de 2011, GitHub anunciou que havia alcançado mais de 1 milhão de usuários.
Em 13 de setembro de 2012, em sua homepage, GitHub anunciou que teve mais de 2,1 milhões de usuários de hospedagem mais de 3,7 milhões de repositórios.
Em 19 de dezembro de 2012, GitHub anunciou que teve mais de 2,8 milhões de usuários de hospedagem mais de 4,6 milhões repositórios.
Em 16 de janeiro de 2013, GitHub anunciou que tinha passado os 3 milhões de usuários marcar e foi então que hospeda mais de 5 milhões de repositórios.
Em 10 de abril de 2013, GitHub anunciaram que tinha 3,5 milhões de usuários e agora hospeda mais de 6 milhões de repositórios.
Em 23 de dezembro de 2013, GitHub anunciaram ter chegado a 10 milhões de repositórios.
